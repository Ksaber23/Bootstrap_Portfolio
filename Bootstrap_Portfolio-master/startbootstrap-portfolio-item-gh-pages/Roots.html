Roots Ancestry Website Software Design Project

Project Description

Roots.xyz was a website designed to track and store consumers ancestry lineage. The website was a project for a masterâ€™s level software design class. The class was split into six groups: team graphical user interface, team quality assurance, team database, team engine, team requirements and team usability. I was placed on team graphical user interface, where I was responsible for developing the front end code for a significant portion of the website. This includes the footers, as well as the pancake menu located on the top right of the picture shown. To collaborate with the other teams, Github was used. We also met as a class, using the Agile management scheme to hold SCRUM meetings that went over our progress developing the website. I also worked closely with team usability, in which I provided them with axure prototypes that they used for user testing. The full functionality of the website has not been realized, but the user interface that was built has significant features. Some of these features were written in Angular JS, which allows quick and responsive feedback to users when signing up to the website.
