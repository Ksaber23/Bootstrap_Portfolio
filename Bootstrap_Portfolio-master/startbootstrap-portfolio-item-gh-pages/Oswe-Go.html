Oswe-Go Research Methods Project

Project Description

Oswe-Go was a proof of concept application designed to reserve parking spots on the SUNY Oswego campus. It was supposed to tackle the problem of finding parking spots without actually having to look manually. First, card sorting was performed to see what type of features the application should have, and where they should be placed. Second, prototyping using Balsamiq and Mockingbird was implemented to design the interface of the application.
