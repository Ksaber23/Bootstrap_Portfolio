TutorTrac Analysis User Interface Evaluation

Project Description

TutorTrac is a website that allows students to schedule tutoring appointments for classes that they are having difficulties with. TutorTrac also handles other functions, such as allowing students to login and out of the system to indicate when they’ve arrived to the appointment and when they’ve left. We used this system in the Office of Learning services at SUNY Oswego, and I had to frequently interact with TutorTrac since I was (and still am) a tutor. In this project, I evaluated the interface, and found several inconsistencies with how the pages were laid out. One of the biggest usability flaws that was found with the website is how the metaphors of certain icons were displayed. To login a student, a user would have to navigate towards the top of the page and input the student’s ID number and hit enter. Unfortunately, the interface displays a magnifying glass in the input field designed to login the student. This same magnifying glass icon was also used in an input box designed to search for classes and tutors. I make an argument in my paper that a different icon metaphor should be used on the login input field.
